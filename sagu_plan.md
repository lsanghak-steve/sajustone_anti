# 사주 기반 천연석 추천 서비스 기획 및 개발 명세서

## 1. 개요
사용자의 이름, 성별, 생년월일 및 태어난 시간을 입력받아 정밀한 사주 오행(나무, 불, 흙, 쇠, 물)을 분석하고, 부족한 기운을 꽉 채워줄 수 있는 맞춤형 실제 천연석과 팔찌를 추천해 주는 세련된 클라이언트사이드 웹 애플리케이션입니다.

---

## 2. 핵심 기능 명세

### 가. 사용자 정보 입력 폼
* **이름 입력**: 맞춤형 결과를 제공하기 위해 사용자의 이름을 입력받습니다.
* **성별 토글 단추**: '🙋‍♂️ 남성 / 🙋‍♀️ 여성'을 고를 수 있는 깔끔한 UI를 제공합니다.
* **생년월일(달력)**: 날짜 선택기를 통해 생년월일을 정확히 지정합니다. (양력 / 음력 토글 지원)
* **태어난 시간 선택**: 사용자가 시간 입력을 누락하더라도 기본값으로 '오전 06:00'가 지정되어 자동 셋팅됩니다.

### 나. 공공데이터 OpenAPI 연동 및 날짜 변환
* **음력 ➡️ 양력 변환**: 음력을 선택하고 분석을 실행하면 한국천문연구원의 `getSolCalInfo` API를 호출하여 실제 양력 날짜를 추출해 사주 분석에 활용합니다.
* **양력 ➡️ 음력 역추적**: 양력을 선택한 경우에도 내부적으로 `getLunCalInfo` API를 호출하여 매칭되는 음력 날짜를 역추적해 옵니다.
* **보안 강화 (.env)**: 서비스 인증키 유출을 막기 위해 UI 설정창을 배제하고, 서버가 켜질 때 `.env` 파일에 기록된 키를 자동으로 읽어와 브라우저 로컬 저장소(LocalStorage)에 탑재하는 백그라운드 처리 방식을 채택했습니다.
* **보안 조치 (.gitignore)**: 인증키 유출을 방지하도록 `.env` 파일을 깃허브 업로드 대상에서 철저히 제외했습니다.
* **CORS 우회 및 예외 처리**: 브라우저 보안 규정(CORS)으로 인한 통신 오류를 차단하기 위해 공용 프록시(`api.allorigins.win`)를 경유하여 데이터를 수신하며, 에러가 발생할 경우 새로고침 없이 사용자에게 알리고 임시 양력 날짜로 사주 연동(Fallback)을 보장합니다.

### 다. 실제 간지 기반 사주 오행 분석
* **실시간 간지 갱신**: 가짜 Mock 데이터를 탈피하고, API가 전송해 준 실제 간지 데이터(세차, 월건, 일진)를 활용해 결과창의 년주, 월주, 일주 한글/한자 카드 정보를 완벽히 갱신합니다.
* **띠 동물 및 오행 속성 동적 판별**: 천간과 지지의 글자를 자바스크립트로 논리 판별하여 해당하는 년/월/일의 띠 동물과 상세 음양 속성 뱃지를 매칭하여 보여줍니다.

### 라. 분석 결과 화면 시각화
* **오행 분포 그래프**: 목, 화, 토, 금, 수의 비율을 시각적인 가로 막대 그래프로 한눈에 볼 수 있도록 시각화하였습니다.
* **운세 풀이 카드**: 2,000자 분량의 정밀 사주 총평 상자와 재물운, 사업운, 연애운, 종합운을 다채롭게 정돈한 4개의 카드 카테고리를 제공합니다.
* **상세 음양력 정보 확인 표**: 양력 날짜(요일 포함), 음력 날짜(윤달/평달 구분), 성별, 태어난 시간, 음력 간지, 율리우스적일을 바둑판식 표로 깔끔하게 띄워줍니다.
* **XML 원본 데이터 뷰어**: 접이식`<details>` 형태의 뷰어를 하단에 배치하여, 실제 공공데이터 서버에서 수신받은 날것의 XML 통신 로그를 클릭 한 번으로 간편하게 볼 수 있도록 제공합니다.

### 마. 맞춤 천연석 추천 및 전체 컬렉션 갤러리
* **오행별 3종 맞춤 추천**: 오행의 흐름을 보완해 주기 위해 오행별로 각 3종씩 매칭된 총 15종의 고유 천연석 데이터를 연동하여, 부족한 기운 발생 시 해당하는 3개의 카드가 나란히 렌더링되게 설계했습니다.
* **실제 이미지 적용**: AI 이미지를 배제하고 직접 촬영되거나 수집된 15종의 실제 고화질 원석 및 팔찌 이미지 경로(원석명_bra.png)를 1:1로 매핑해 보여줍니다.
* **구매 링크 연동**: 이미지 클릭 시 새 탭(`target="_blank"`)으로 쇼핑몰 구매 페이지가 열리도록 구성하였으며, 호버 시 밝아지는 직관적 효과를 줍니다.
* **독립된 갤러리 페이지 (`stones.html`)**: 메인 화면뿐만 아니라 15종의 천연석을 오행별 섹션(목, 화, 토, 금, 수)으로 정돈하여 감상할 수 있는 구경하기 전용 격자 갤러리 페이지를 제공합니다.

## 3. 기술 스택
* **HTML5 / CSS3**: 명조 및 고딕이 어우러진 깔끔한 Noto Sans KR 폰트, 모바일 반응형 그리드 시스템 레이아웃.
* **Vanilla JavaScript (ES6+)**: 별도의 프레임워크 없이 웹 기본 Fetch API, DOM 파서, 비동기(async/await) 로직 활용.
* **데이터 API**: 한국천문연구원 OpenAPI (CORS 프록시 우회).
* **보안 환경**: `.env` 및 `.gitignore`.

## 4. 최종 통합 개발 이정표 (성과)

1. **[완료] 기초 화면 & 레이아웃 설계**: 이름/성별/생년월일 입력 및 오전 시간 기본값(06:00) 설정 등 모바일 친화적 입력 폼 UI 완성.
2. **[완료] 분석 결과 시각화**: 사주 4기둥 카드 레이아웃, 오행 비율 가로 막대 그래프 시각화 및 방대한 2000자 운세 카드 완성.
3. **[완료] 천연석 15종 실제 데이터 구축**: 부족한 기운 발생 시 3종 카드 동적 생성, 실제 고화질 원석/팔찌 이미지 매핑, 구매 링크 연결 및 `stones.html` 갤러리 개설.
4. **[완료] 공공데이터 API 실제 연동**: 음양력 양방향 변환 기능과 CORS 차단 해결, 세차/월건/일진 실제 데이터 연동 및 사주 4기둥/띠 동적 갱신 기능 탑재.
5. **[완료] 보안 강화 및 XML 뷰어 도입**: 화면 상의 복잡한 키 설정 UI를 제거하여 보안성 확보, 글로벌 에러 방어막(Try-Catch) 구축으로 새로고침 차단 및 하단 접이식 XML 뷰어 연동.
6. **[완료] 명리학 상세 데이터 계산 엔진 및 결과 UI 개편**:
   * 입력된 시간에 따른 **시주 실시간 계산(시두법)** 함수 구현.
   * 사주 8자의 오행(목, 화, 토, 금, 수)을 실시간으로 계산하여 오행 분포 그래프에 동적 반영.
   * 일간 기준 타 기둥의 **십성 및 12운성** 판정 알고리즘 추가.
   * 대운(시작나이, 대운간지, 12운성 등), 세운, 월운 리스트 계산기 및 지장간/신살 판별 정보 UI 연동.
   * 결과 화면에 "사주 4기둥(시주-일주-월주-연주)", "십성/12운성 상세 테이블", "대운/세운/월운 테이블" 영역을 추가하고 고품격 UI로 렌더링.
7. **[완료] CORS 프록시 서버 이관 및 캐시 무력화 개편**:
   * 불안정하고 접속 차단이 빈번하던 `api.allorigins.win` 프록시 서버를 대신하여 브라우저에서 안정적인 `corsproxy.io` 중계기로 교체 적용.
   * XML 텍스트 수신 방식으로 통신 성능을 개선하고, `&cb=Date.now()` 타임스탬프 난수를 덧붙여 중간 중계 서버의 캐싱 오류 차단.
   * 보안인증서 통신 차단을 우회하기 위해 호출 프로토콜을 `http`로 조정하여 실시간 음/양력 변환 무오류 보장.
8. **[완료] 음력 날짜 변환 오류 대응 및 HTTPS 주소 복원**:
   * 브라우저의 보안 정책(Mixed Content)으로 인한 통신 차단 및 프록시 서버의 HTTP 지원 한계를 극복하고자 API 호출 주소의 프로토콜을 `http`에서 `https`로 안전하게 원복함.
   * 날짜를 자릿수에 맞춰 정렬할 때 문자열 타입이 보장되지 않아 발생할 수 있는 런타임 오류를 방지하기 위해 `String()` 형변환 방어 코드를 적용함.
