# 웹사이트 기획안: 사주 기반 천연석 추천 서비스

## 1. 개요
사용자의 생년월일을 입력받아 사주(오행: 나무, 불, 흙, 쇠, 물)를 분석하고, 부족한 기운을 채워줄 수 있는 천연석(원석)을 추천해주는 웹사이트입니다.

## 2. 주요 기능
- **생년월일 입력 폼**: 연도, 월, 일, (선택) 태어난 시간을 입력할 수 있는 화면.
- **사주 분석 (오행 계산)**: 입력된 생년월일을 바탕으로 오행(목, 화, 토, 금, 수)의 비율을 간단히 계산하여 부족한 기운을 판별합니다.
- **천연석 추천**: 부족한 기운(예: 불이 부족함)에 맞는 천연석(예: 자수정, 루비 등)의 이름, 사진, 의미를 결과로 보여줍니다.

## 3. 디자인 방향
- **심플하고 깔끔한 UI**: 시각적으로 피로하지 않은 색상을 사용합니다. 하얀색 배경에 부드러운 파스텔톤으로 포인트를 줍니다.
- **모바일 친화적 (반응형)**: 스마트폰이나 태블릿에서도 입력과 결과를 보기 쉽도록 크기가 자동 조절됩니다.
- **읽기 쉬운 글씨**: 명조체나 깔끔한 고딕체를 사용하여 신뢰감을 줍니다.

## 4. 기술 스택
- **HTML/CSS/JavaScript**: 복잡한 프레임워크 없이 웹의 기본 기술을 사용하여 가볍고 빠르게 만듭니다.

## 5. 단계별 개발 계획
1. **1단계**: 기획안(plan.md) 작성 (완료)
2. **2단계**: 웹사이트의 뼈대(HTML)와 디자인(CSS) 만들기 (입력 폼 화면 구성)
3. **3단계**: 사주 분석 및 천연석 추천 기능(JavaScript) 연결하기
4. **4단계**: 결과 화면 디자인 다듬기 및 전체 테스트

## 6. 추가 업데이트: 상세 사주 결과 UI 개편 (진행완료)
- **이름 입력칸 추가**: 결과를 '[이름]님의 오행 결과'로 맞춤형으로 표시합니다.
- **사주 4기둥 표시**: 년주, 월주, 일주, 시주를 한자와 한글, 띠와 함께 전문적으로 디자인하여 배치했습니다.
- **오행 분포 그래프**: 목, 화, 토, 금, 수의 비율을 시각적인 막대 그래프로 한눈에 볼 수 있도록 시각화했습니다.
- **가짜 데이터(Mock) 적용**: 복잡한 실제 만세력 계산 대신, 화면 디자인과 작동 방식을 완벽하게 보여주기 위해 임의의 고정된 사주 데이터를 활용하여 화면을 완성했습니다.

## 7. 대규모 운세 풀이 텍스트 확장 (진행완료)
- **사주 총평 추가**: 약 2000자 분량의 깊이 있고 희망찬 전체 사주 풀이를 넓고 편안한 텍스트 박스로 제공합니다.
- **분야별 올해의 운세**: 재물운, 사업운, 연애운, 종합운을 각각 500자 분량으로 작성하여, 4개의 예쁜 카드 형태로 배치하여 가독성을 높였습니다.
- **천연석 심층 분석**: 기존의 짧은 추천사를 넘어, 각 오행이 왜 필요한지 500자 분량으로 아주 상세하게 설명하는 심층 분석 영역을 추가했습니다.

## 8. 천연석 팔찌 이미지 추가 (진행완료)
- **AI 이미지 생성**: 각 오행(목, 화, 토, 금, 수)에 맞는 아름답고 고급스러운 원석 팔찌 이미지 5종을 인공지능으로 새롭게 생성하여 적용했습니다.
- **화면 레이아웃 변경**: 기존의 원석 사진 영역 바로 아래에 '팔찌 착용 샷'이 추가로 나타날 수 있도록 화면 구조(`index.html`, `style.css`)와 데이터를 업데이트했습니다.
